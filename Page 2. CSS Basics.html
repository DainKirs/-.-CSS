Основы CSS

Подключение CSS к HTML

Далее речь пойдет о практических основах CSS. Начало работы с таблицами стилей заключается в выборе способа их подключения, как было сказано ранее есть три основных способа:

  1. Подключение через атрибут style: Вы можете добавить CSS-свойства непосредственно в тег HTML, используя атрибут style.
  2. Подключение через внутренние ссылки на CSS-секции: Вы можете определить CSS-секции внутри тега HTML и использовать их для внедрения стилей.
  3. Подключение через внешние файлы CSS: Вы можете создать отдельный файл CSS и подключить его к вашему HTML-документу с помощью тега link.

Именно последний способ является наиболее популярным на сегодняшний день.

И так вам необходимо создать файл с расширением .css, например style.css. Для удобства данный файл желательно разместить, в той же папке, что и остальные html файлы. Чтобы связать html файл с css необходимо использовать тег link. Пример: link rel="stylesheet" href="Style.css". Данный тег необходимо прописать внутри тега head. После чего, ваш html код будет обращаться к css файлу, для установки оформления стринц.

Свойства и назначение свойств

Свойства в CSS это основной параметр, засчет которого имеется возможность назначать любые визуальные стили, формируя таким образом уникальный внешний вид сайта.

Существует множество свойств CSS, каждое из которых отвечает за определенный аспект оформления. Некоторые из них

Отображение на экране. По умолчанию элементы делятся на строчные и блочные и разница в них только в том, что блочные не могут стоять в одной строке друг с другом. Зато строчным нельзя задать размер, он зависит от того, сколько в них содержимого. Итак, иногда такое поведение нужно изменить и для этого используется display и множество его значений, но я расскажу лишь о самых основных:

Block – элемент становится блочным (то есть приобретает блочные особенности, даже если он был строчным. С этих пор ему можно записывать размеры, нормально определять внешние отступы и т.д.)

Inline- преобразование в строчный элемент.

Inline-block – блочно-строчный тип. Это такое поведение, при котором элемент сохраняет свои блочные свойства, но при этом может стать в одну строку с другими блоками, если им тоже прописано inline-block и им хватает места по ширине. Стоит помнить, что по умолчанию блок занимает в родителе все свободное место по ширине, потому нужно вручную определить ширину для него (абсолютную или относительную).

Table — преобразование элемента в таблицу (соответственно, есть также значения table-row и table-cell)

None – элемент исчезает со страницы и не оставляет никаких следов, как будто бы его на ней и не было.

Опять же, это не все значения, только самые основные.

Управление размером

Блоки играют основную роль в создании сайта. Им нужно прописывать определенные размеры, это указывается с помощью таких свойств:

  1. Width – ширина
  2. Height – высота

Данные параметры можно указывать в разных единицах измерений, самая популярная из них - пиксели (px), так-же часто используют процентное соотношение, относительно размеров окна. Max-width, min-width – максимальная или минимальная ширина. Например, если указать блоку max-width, то эта ширина не будет жесткой, она сможет меняться в зависимости от размеров окна. Если же указать и min-width, то тогда блок не сможет стать уже, чем указанный размер. Такое же можно прописать и для высоты, но это делают гораздо реже. Также на размеры блочных элементов могут повлиять дополнительные параметры. Например, внутренние отступы. Они задаются с помощью слова padding и значения в пикселях (но можно и в другой величине). Если нужно задать отступ только для одной стороны, можно указать это, дописав к padding- через дефис нужную сторону (left, right и т.д.) Например, если мы определили ширину контейнера в 600 пикселей, а потом добавили внутренние отступы по бокам в 20 пикселей, то реальная новая ширина составит 640px. Если вы не хотите, чтобы размеры добавлялись, можно воспользоваться таким приемом:

*{

box-sizing: border-box;

}

1

2

3

*{

box-sizing: border-box;

}

В итоге теперь, когда вы будете определять ширину через width, в нее автоматически будут включаться внутренние отступы и рамка. Рамку задают с помощью параметра border. Ее тоже можно определить как для всех сторон, так и для каждой отдельно. Синтаксис таков: border: ширина стиль цвет; Соответственно, записав так:

#{

border: 3px solid blue;

}

К блокам добавляется синяя рамка со всех сторон толщиной 3 пикселя с каждой. Суммарно ширина увеличивается на 6 пикселей, а если вы не хотите этого, нужно использовать то самое box-sizing.

Цвет и фон

Это основные параметры, которые отвечают за восприятие нами информации. В значении в этим свойствам записывается одно и то же – цвет, который можно определить с помощью ключевых слов, в шестнадцатеричном коде или каком-то цветовом формате (чаще всего rgb). Единственное отличие – цвет определяется с помощью свойства color, а фон – с помощью background-color.

Свойство background позволяет задать фоновое изображение или цвет для элемента на веб-странице. Для этого нужно указать URL-адрес изображения или установить цвет с помощью шестнадцатеричного кода. Например, для задания фона с помощью изображения можно использовать следующий код:

body {

background-image: url('image.jpg');

}

А для задания цвета можно использовать следующий код:

body {

background-color: #FFFFFF;

}

Свойство color позволяет задать цвет текста на веб-странице. Для этого нужно указать цвет с помощью шестнадцатеричного кода. Например, для задания белого цвета текста можно использовать следующий код:

body {

color: #FFFFFF;

}

Селекторы

Далее речь пойдет о селекторах.

Селекторы — это шаблоны, которые используются для привязки стилевых свойств к элементам в документе. Вы можете задать стиль для всех элементов или сократить выбор с помощью определённого селектора. Селектор выбирает элемент(ы) для применения стиля (в данном случае, элементы p ). Для стилизации другого элемента, просто измените селектор. Селекторы типа выбирают элементы HTML-документа по их тегу. Например, селектор p выберет все <p> на странице:

HTML

<body>

<p>Текст</p>

<p>Другой текст</p>

</body>

CSS

p {

font-size: 16px;

}

Селекторы класса определяют стиль элементов с определённым классом. Перед такими селекторами необходимо ставить точку, чтобы указать, что речь идет о выборе класса. Например, этот селектор выберет абзац с классом .highlight.

HTML

<body>

<p class="highlight">Текст</p>

<p>Другой текст</p>

</body>

CSS

.highlight {

background-color: yellow;

}

Селекторы идентификатора выбирают элемент по его уникальному идентификатору. У таких селектоов в начале ставится #. Например, #welcome соответствует элементу с id="welcome":

HTML

<div id="welcome">

<p>Добро пожаловать!</p>

</div>

CSS

#welcome {

background-color: blue;

}

Селекторы наличия и значения атрибута находят все элементы, которые имеют определённый атрибут или значение атрибута. Например, здесь мы выбираем все ссылки, которые начинаются на http://, но не имеют example.com.

a[href^="http://"]:not([href*="[example.com](http://example.com/)"]) {

color: #000;

text-decoration: none;

}

Универсальный селектор позволяет прописать стили для всех элементов HTML без исключения. Он указывается в виде звёздочки *. В примере ниже мы используем универсальный селектор с псевдоэлементами, чтобы изменить расчёт общего размера элемента.

*,

*::before,

*::after {

box-sizing: border-box;

}

Существует множество различных типов селектора. Выше мы рассматривали только селектор элементов, который выбирает все элементы определённого типа в HTML документе. Но мы можем сделать выбор более конкретным. Вот некоторые из наиболее распространённых типов селекторов:

Селектор элемента (иногда называемый селектором тега или типа) Все HTML-элемент(ы) указанного типа. p Выбирает < p >
ID селектор Элемент на странице с указанным ID на данной HTML. Лучше всего использовать один элемент для каждого ID (и конечно один ID для каждого элемента), даже если вам разрешено использовать один и тот же ID для нескольких элементов. #my-id Выбирает < p id="my-id"> или < a id="my-id">
Селектор класса Элемент(ы) на странице с указанным классом (множество экземпляров класса может объявляться на странице). .my-class Выбирает <p class="my-class"> и <a class="my-class">
Селектор атрибута Элемент(ы) на странице с указанным атрибутом. img[src] Выбирает < img src="myimage.png"> но не <img >
Селектор псевдокласса Указанные элемент(ы), но только в случае определённого состояния, например, при наведении курсора. a:hover Выбирает <a>, но только тогда, когда указатель мыши наведён на ссылку.

Обратите внимание на важные части синтаксиса:

Каждый набор правил (кроме селектора) должен быть обёрнут в фигурные скобки ({}). В каждом объявлении необходимо использовать двоеточие (:), чтобы отделить свойство от его значений. В каждом наборе правил вы должны использовать точку с запятой (;), чтобы отделить каждое объявление от следующего. Таким образом, чтобы изменить несколько значений свойств сразу, вам просто нужно написать их, разделяя точкой с запятой, например так:

p { >

color: red;

width: 500px;

border: 1px solid black;

}

Выбор нескольких элементов

Вы также можете выбрать несколько элементов разного типа и применить единый набор правил для всех из них. Добавьте несколько селекторов, разделённых запятыми. Например:

p,li,h1{

color: red;

}

Анимация и ключевые кадры

Css позволяет не только создавать красивое оформление для страницы, но и анимировать элементы на данной странице. CSS-анимации позволяют анимировать переходы от одной конфигурации CSS стилей к другой. CSS-анимации состоят из двух компонентов: стилевое описание анимации и набор ключевых кадров, определяющих начальное, конечное и, возможно, промежуточное состояние анимируемых стилей.Что такое ключевой кадр? В программах анимации, CSS и большинстве других инструментов, позволяющих анимировать что-либо, ключевые кадры — это механизм, который используется для назначения состояний анимации временным меткам, расположенным на временной шкале.

Так, на примере ниже прописана анимация для наведения курсора на ссылку, при которой она будет исчезать и появляться. По ключевым кадрам видим, что в начале анимации (0%) объекту присвоена полная прозрачность, а по завершению (100%) он полностью непрозрачный.

a:hover {

animation: fadeIn 3s ease-in-out forwards;

}

@keyframes fadeIn {

0% {

opacity: 0;

}

100% {

opacity: 1;

}

}

Вот наглядный пример того, как это должно выглядеть.

Анимации CSS не ограничиваются исчезновением текста. При помощи анимаций можно задавать изменение размера и траекторию движения объектам. Например следующий код позволит создать два круга, один из которых будет пульсировать, а второй летать из стороны в сторону.

HTML

<div class="container">

<div class="circle pulse"></div>

<div class="circle fly"></div>

</div>

CSS

.container{

background-color: rgb(255, 255, 255);

display: flex;

justify-content: center;

align-items: center;

height: 100vh;

}

.circle {

width: 150px;

height: 150px;

border-radius: 50%;

background-color: rgb(253, 10, 10);

position: relative;

}

.pulse {

animation: pulse 1s infinite alternate;

}

.fly {

animation: fly 2s infinite alternate;

}

@keyframes pulse {

from {

transform: scale(1);

}

to {

transform: scale(2);

}

}

@keyframes fly {

0% {

left: 0;

}

50% {

left: calc(100vw - 50px);

}

100% {

left: 0;

}

}
