Электронные ресурсы

1. Developer Mozilla - ресурс для разработчиков, на котором можно найти много полезной информации по CSS

2. Хабр - русскоязычный форум по тематике IT

4. Flexbox Froggy - ресурс, позволяющий в игровой форме освоить Flexbox

5. Codecademy — курсы и обучающие материалы по CSS

6. CSSTricks — уроки, советы и трюки для становления экспертом CSS

7. Stepik - каталок электронных курсов на различные тематики, включая HTML и CSS

8. Code.mu - электронный справочник для разработчиков и верстальщиков

9. Github - сообщество разработчиков предоставляющее бесплтаный хостинг, портфолио и систему контроля версий, является весьма полезным инструментом для любого разработчика

Так же не забывайте, что многие используют видеохостинг и стриминговые серивисы для публикации бесплатных обучающих курсов в сфере разработки, включая и веб-разработку тоже.
